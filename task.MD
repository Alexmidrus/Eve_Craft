В моделу SDE необходимо исправить логгику загрузки изображений:
1. Сейчас это статичный адрес для загрузки. Необходимо собирать адрес динамически, на случай изменения. По этой ссылке публикуются ссылки на архивы с файлами. https://raw.githubusercontent.com/esi/esi-docs/main/docs/services/iec/index.md, нужно брать ссылку отсюда.
2. Если удалить базу SDE, но оставить фото. То весь процесс загрузки будет поторен как для SDE так и для фото. Тоже самое для фото если удалить папку сресурсами то JsonL тоже будет повторно загружен. Нужно проверять таргетно если нет базы но есть фото грузим базу, если нет фото но есть база грузим фото, если ничего нет то грузим все. Если все загружено, то запускаем програму
